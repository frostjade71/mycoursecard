## Features

### Course-Aware Tailoring
- **Smart Onboarding**: Select your course and instantly get a pre-configured template designed for your specific career path.
- **Diverse Support**: Tailored sections for Nursing clinical hours, CS GitHub projects, Business case studies, and more.

### Intelligent Builder
- **Drag-and-Drop Ordering**: Effortlessly reorder your sections using an intuitive interface powered by @dnd-kit.
- **Rich Content**: Showcase projects, certifications, skills, and experiences with a clean, minimal design system.

### Professional Presence
- **One-Click Preview**: Enter your details and see your professional presence come to life in real-time.
- **Public Portfolio URL**: Get a shareable `mycoursecard.app/u/[username]` link to send to recruiters and professors.

### Tooling & Accessibility
- **PDF Resume Export**: Automatically generate a polished, one-page PDF resume from your portfolio data.
- **Glassmorphic UI**: A premium, responsive design that looks stunning on every device, from mobile to desktop.

## Technology Stack

- **Frontend Core**: React 19, Vite 8, TypeScript, React Router 7
- **Database & Auth**: Supabase (PostgreSQL, Googl OAuth)
- **Styling**: Tailwind CSS 4 (Stitch-inspired palette)
- **Interactive UI**: @dnd-kit (Sortable)
- **Export Engine**: jsPDF & html2canvas
- **Icons**: Lucide React
- **Deployment**: Vercel
